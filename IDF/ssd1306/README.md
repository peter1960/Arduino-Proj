# _Sample project_

Given:
Diameter = 700 mm = 0.7 m
Speed = 40 km/h

Step 1: Calculate circumference
Circumference = π * Diameter
Circumference ≈ 3.1416 * 0.7
Circumference ≈ 2.199 m

Step 2: Convert speed to m/s
Speed = 40 km/h = (40,000 m) / (3600 s)
Speed ≈ 11.11 m/s

Step 3: Time for one rotation
Time = Circumference / Speed
Time ≈ 2.199 / 11.11
Time ≈ 0.198 s per rotation

Step 4: Calculate RPM
Rotations per second = 1 / 0.198 ≈ 5.05
RPM = 5.05 * 60 ≈ 303 RPM
